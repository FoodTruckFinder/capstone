Conceptual Model

Entities & Attributes

User
  • userId (primary UUID)
  • userEmail
  • userActivationToken
  • userName
  • userHash
  • userIsOwner
Favorite
  • favUserId (foreign UUID)
  • favTruckId (foreign UUID)
  • favAddDate
Like
  • likeIsUserId (foreign Uuid)
  • likeIsTruckId (foreign Uuid)
  • likeDate
Foodtruck
  • foodTruckId (primary key Uuid)
  • foodTruckUserId (foreign key Uuid)
  • foodTruckName
  • foodTruckRegistrationNumber
  • foodTruckImageUrl
  • foodTruckMenuUrl
  • foodTruckDescription
  • foodTruckPhoneNumber
  • foodTruckSocialMediaUrlLink
Location
  • locationFoodTruckId (primary Uuid)
  • locationUserId (foreign Uuid)
  • locationLat
  • locationLong
  • locationFoodTruckDate
  • locationDuration
Relationships
  • Many truck profiles can post many locations
  • Many users can like many foodtruck profiles
  • Many truck profiles can own many locations
  • One user can like many foodtrucks (One like per one foodtruck)
  • One user owner to one foodtruck profile
