User Story

Planning a night out at the bar and wonder if there will be any good food trucks in the area.

Use Case

Title: Looking for food trucks downtown.

Description: Chad gets on google to find food trucks near me.

Goals:Chad wants to find foodtrucks when he goes out and wants to be able to see pricing, location, and menu options all in one convenient location.

Name of User & Their Role: Chad Garcia; potential food truck customer.

Usage Preconditions:Has to have a mobile phone capable of displaying the food truck locator website.

Usage Postconditions:The website confirms that the food truck that Chad likes will be at the brewery he is going to.

Interaction Flow:

  • Chad enters 505FoodTruckFinder.com into his iPhone 8 mobile firefox browser.
  • Server receives Chad's request and returns landing page for 505FoodTruckFinder.com.
  • Chad enables location services to see nearby foodtrucks within x distance.
  • Server receives request and returns map data of foodtrucks within distance specified based on Chad's current location.
  • Chad receives map with pinned locations of food trucks that are in the area.
  • Chad clicks on one of the pins on map belonging to an active foodtruck.
  • Server returns profile page of specific pinned truck.
  • Chad finds menu link on profile page and clicks to view.
  • Server receives request for menu photo and loads file to Chad's browser.
  • Chad reviews menu and confirms that they have the desired items.
  • Chad clicks a back button to navigate back to the selected foodtruck's profile page which shows a small map and a pin of the trucks current location.
  • Chad confirms that the selected Foodtruck is in his desired location.
  • Chad completes foodtruck search and bookmarks 505FoodTruckFinder on his phone and adds icon to home screen.

Frequency of Use: Twice a week
